CONFIGURACION FIREBASE

Para sincronizar datos entre el entrenador y el atleta necesitas configurar Firebase (gratis). Solo se hace una vez.

1
Crear proyecto Firebase
Ve a console.firebase.google.com → "Crear proyecto" → ponerle un nombre (ej: forge-pilot)
2
Crear base de datos
En el menu lateral: Realtime Database → "Crear database" → elegir "Modo de prueba" (permite lectura/escritura durante 30 días)
3
Obtener configuracion
Configuracion del proyecto (icono engranaje) → "Tu app" → agrega una app Web → copia el objeto firebaseConfig
4
Pegar aqui
Abre este archivo HTML en un editor de texto, busca PEGAR_FIREBASE_CONFIG_AQUI y reemplaza el objeto con tu configuracion
