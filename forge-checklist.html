📱 LA APP Crítico
✓
Test completo del flujo entrenador → atleta
Abrí la app en dos pestañas. En una entrá como Entrenador, creá un mesociclo básico. En la otra entrá como Atleta con el código. Verificá que el entrenador vea al atleta y pueda asignarle el mesociclo.
✓
Test desde celular (Android o iPhone)
Abrí la URL de Netlify desde tu celular. Chequeá que el formulario de sesión sea usable, que los inputs sean cómodos y que el semáforo se vea bien en pantalla chica.
✓
Verificar que el importador XLSX funciona
Descargá la plantilla desde la app, completá 2-3 filas y subila. Verificá que crea el mesociclo correctamente con las sesiones y ejercicios.
✓
Simular una sesión completa como atleta
Iniciá una sesión, completá el check de bienestar, cargá datos en todos los ejercicios y finalizá. Verificá que aparece el confetti, que el semáforo se actualiza en el dashboard del entrenador y que el log queda guardado en Firebase.
✓
Verificar persistencia — cerrar y reabrir
Después de la sesión anterior, cerrá el navegador completamente. Volvé a abrir la URL. Verificá que el entrenador entra directo al dashboard (sin pedir login) y que los datos de la sesión siguen visibles.
✓
Limpiar datos de test después de verificar
Volvé a Firebase → Data y borrá el nodo forge nuevamente para que el entrenador real arranque con la base limpia.
🌐 HOSTING Importante
✓
Subir los 3 archivos juntos a Netlify
Creá una carpeta con los tres archivos: forge-firebase.html, forge-landing.html y forge-feedback.html. Arrastrá la carpeta (no los archivos sueltos) a drop.netlify.com. Los links internos funcionan entre ellos.
Abrir Netlify Drop →
✓
Verificar que la landing abre correctamente
Abrí la URL de Netlify. La página de inicio debe ser la landing (forge-landing.html). El botón "Ver demo" debe llevar a forge-firebase.html.
✓
Anotar la URL definitiva
Netlify genera una URL tipo random-name-123.netlify.app. Anotala — esa es la que le vas a mandar al entrenador. Si querés una URL más prolija, podés customizarla gratis en la config de Netlify (ej: forge-pilot.netlify.app).
💬 QUÉ DECIRLE Importante
El mensaje para el entrenador. Copiá y adaptá este texto:
"Hola [nombre]! Te paso el prototipo de la app de entrenamiento que te mencioné.

Cómo usarla:
1. Abrí este link: [URL]
2. Elegí 'Entrenador', poné tu nombre → te aparece un código de 6 letras
3. Ese código se lo das a tus atletas para que se vinculen
4. Tus atletas abren el mismo link, eligen 'Atleta', ingresan el código y su nombre
5. Desde tu dashboard asignás el mesociclo a cada atleta

Contexto: Es un prototipo funcional, no una app terminada. Los datos se guardan bien y no se pierden, pero el sistema no tiene toda la seguridad de producción todavía. Usalo con atletas de confianza.

Si algo no funciona o tenés sugerencias, avisame por acá. En un mes te mando un formulario corto para saber qué te pareció."
✓
Preparar el mensaje para el entrenador

✓

✓
Agendar envío del formulario de feedback al mes
